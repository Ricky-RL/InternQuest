What tech stack should we use?

This is a frontend-only MVP with hardcoded data — no backend, no API.

A

React + Vite (Recommended)

Component-based, fast dev server, easy to deploy to Vercel/Netlify. Industry standard for modern SPAs. Great for a job board with filters and cards.

B

Next.js

React framework with file-based routing. More structure out of the box. Slight overkill for a pure frontend MVP but easy to add a backend later.

C

Plain HTML + Tailwind CSS

No framework. Fastest to scaffold, easiest to understand. Less flexible for dynamic filtering and interactivity but totally viable for MVP.
