Which pages should the MVP include?

Select all that apply — we can always cut scope later.

1

Job Board (main feed)

The core page — filterable list of internship postings with search, category filters, and job cards.

2

Job Detail Page

Full description of a single posting — responsibilities, requirements, company info, apply button.

3

User Profile / Resume Upload

Lets the user enter skills, experience, and upload a resume — sets up future AI matching. Hardcoded/mock state for MVP.

4

Landing / Marketing Page

Hero section, value props, CTA to sign up or browse jobs. Good for pitching the product.
